1. Install virualenv
    MAC -   python3 -m pip install virtualenv
    WIN -   py -m pip install virtualenv
    -- using python i want to use the modules, the one that i want to use is the pip, and i want to install virtualenv

2. Use venv to create a virtual enviroment
    MAC -   python3 -m venv venv
    WIN -   py -m venv venv

3. We need to launch the virtualenv

    MAC -  source venv/bin/activate
    WIN -  venv\Scripts\activate

4. we need to install flask - this is the server library

    MAC -   python3 -m pip install flask
    WIN -   py -m pip install flask
